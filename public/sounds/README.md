# Audio Files Directory

Place your audio files here according to the instructions in `/AUDIO_SETUP.md`

## Required Files:

- `ambient.mp3` - Background ambient sound (looping)
- `habibie.mp3` - B.J. Habibie iconic sound (3-8 seconds)
- `gus-dur.mp3` - Gus Dur iconic sound (3-8 seconds)
- `megawati.mp3` - Megawati iconic sound (3-8 seconds)
- `sby.mp3` - SBY iconic sound (3-8 seconds)
- `jokowi.mp3` - Jokowi iconic sound (3-8 seconds)
- `prabowo.mp3` - Prabowo iconic sound (3-8 seconds)

## Note:

The website will work perfectly fine without these audio files. They are optional enhancements.
If files are missing, the audio system will gracefully fail without breaking the user experience.
